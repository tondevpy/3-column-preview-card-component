# Componente de Cartões de Visualização de 3 Colunas
Este é um projeto de um componente de cartões de visualização de 3 colunas, desenvolvido como uma solução para o desafio "3-column preview card component" do Frontend Mentor.

# Descrição do Projeto
Este projeto consiste em um layout responsivo de 3 cartões de visualização, cada um apresentando uma categoria de veículo: Sedans, SUVs e Veículos de Luxo. Cada cartão exibe uma imagem representativa da categoria, um título, uma breve descrição e um botão "Learn More".

# Tecnologias Utilizadas
HTML
CSS
